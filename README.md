# Vivek Pandey • Full Stack Developer & AI Engineer

Welcome to my personal portfolio! This is a modern, premium, and fully responsive portfolio website implemented using **Next.js**, **Framer Motion**, **GSAP**, and **Tailwind CSS**. 

## 🌐 Live Site
You can visit the live website here: [vivek.technewity.com](https://vivek.technewity.com/)

---

## 🔧 Tech Stack
* **Framework**: Next.js (App Router)
* **Animation**: Framer Motion & GSAP (with Lenis smooth scrolling)
* **Styling**: Tailwind CSS & Styled Components
* **Asset Optimization**: Next-Cloudinary
* **Linting & Formatting**: ESLint, Prettier, Stylelint, Husky, and Lint-staged

---

## ✨ Features
* **Custom Favicon**: Custom `VP` monogram brand favicon.
* **Smooth Page Transitions**: Custom curtain preloader animation displaying page names on load and page transition.
* **Responsive Layouts**: Designed to look premium and render beautifully on all device viewports.
* **Interactive Elements**: Magnetic buttons, text scroll reveals, and custom animations.
* **Live Local Time**: Integrates a dynamic local clock and app versioning on all layout footers.
